LiDAR Lense

My iPhone app that used the LiDAR on some iPhone Pro models to help map the room around you. It identifies potential obstacles and displays them on the screen for the user as they walk around real time. Written in Swift and Python, the repository is currently private.

Image Lab

Image Lab is a fast, browser-based image editor with a Python (Flask) backend. It supports cropping, resizing, color adjustments, filters, background removal, histogram equalization, and content-aware seam carving.

Music Aggregator for DJing

This is a GUI I wrote that will take in a spotify playlist and download that playlist as a series of MP3s, all tagged with standard ID3 tags. This makes importing into DJ software, such as Serato, trivial. It is highly parallelized, being able to support downloading 6 playlists at a time.

First Person View Drone

Senior design project for ECE at Illinois. 3 custom designed PCBs used to implament live wireless camera transmission from a Crazyflie drone to a headset. Done in a group of 3. The final drone successfully streamed real time video to both a laptop and a headset.

Repository

Six Degrees of Wikipedia Solver

A program to solve the six degrees of Wikipedia problem, which is described here. It was a final project, done in a group of 4. It also goes into the betweenness centrality of thousands of articles.

Repository

Custom Video Game on FPGA

A custom video game programmed onto an FPGA detailing the life of an ECE major @ U of I. Written in System Verilog it includes custom sprites and top down gameplay. The repository is currently private.

Controlled Boost Converter

A hand-controlled boost converter designed for a ranging 9-15V input, and a strict 24V±3% output. A custom inductor was wound for the project. Arduino control was also built in for more precise control.

Demonstration of Seam Carving using Manim

A final project in my Image Engineering class where I use the community version of Manim to explain in detail how seam carving works. I didn't have a good microphone so the audio is not great.
